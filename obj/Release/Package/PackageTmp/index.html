Call for Speakers

The TECHbash planning committee is now accepting presentation submissions for the 2013 TECHbash conference. This year's conference includes four available tracks for all attendees:

  • Web Development
  • Mobile Application Development
  • Enterprise IT
  • Professional Growth (startups, consulting)
  • Other

If you are interested in submitting a presentation, please send an email to: speak@techbash.com with the title and summary of your presentation, track category, audience skill level, a self-bio, and a favorite image for your avatar. Presentations are limited to 1 hour.
